# 🌐 Portfolio Website - Chaesarrio Taufikul Hakim

Website portofolio pribadi yang menampilkan profil, keterampilan, pengalaman, dan sertifikat. Dibuat dengan **HTML dan CSS murni** 

---

## 🛠️ Teknologi yang Digunakan

| Teknologi | Keterangan |
|-----------|------------|
| **HTML5** | Struktur dan konten website |
| **CSS3** | Styling, layout, dan animasi |
| **Font Awesome** | Icon (via CDN) |
| **Google Fonts** | Font Poppins (via CDN) |
